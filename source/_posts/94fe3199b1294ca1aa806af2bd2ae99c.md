**快捷键**

| 功能           | Mac 键盘按键             | Windows 键盘按键  |
| ------------ | -------------------- | ------------- |
| 删除光标前 整行字符   | Command (⌘) + Delete | Ctrl + Delete |
| 删除光标后 第一个字符  | Control (⌃) + D      | Win + D       |
| 删除光标到行末 所有字符 | Control + K          | Win + K       |

***

**Windows 键盘与 MacBook Pro 的“键位位置”映射**

| Windows 键盘按键 | 对应 Mac 功能键  | 映射后 Windows 键的作用 |
| ------------ | ----------- | ---------------- |
| Ctrl         | command (⌘) | 触发 Command 功能    |
| Alt          | option (⌥)  | 触发 Option 功能     |
| Win          | control (⌃) | 触发 Control 功能    |

打开 系统设置 > 键盘 > 键盘快捷键… > 修饰键。

1. 在顶部“选择键盘”下拉菜单中，选择连接的 Windows 键盘。
2. 在下方弹出的菜单中，为每个“修饰键”重新分配功能。
3. 如果选择上述的“键位位置映射”，则将“Control (⌃) 键”改为“Command”，将“Command (⌘) 键”改为“Control”。“Option 键”通常无需改动。
